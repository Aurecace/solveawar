The second conditional

What is the second conditional?

Do you remember the first conditional? We used the first conditional to speak about consequences of a present situation.

Example: If everyone has weapons, Spain will be an unsafe country.

However, we use the second conditional to speak about the consequences of an imaginary present or future situation.

Example: If all the people had weapons, Spain would be an unsafe country.

Formation of second conditional

To create sentences using the second conditional, we need to combine the following parts:

  1. An imaginary action. We use the word "IF" followed by any imaginary action you want to speak about in PAST SIMPLE.
    Structure IF an imaginary action in PAST SIMPLE
    Example If my parents didn't have a car...
  2. A consequence that we think will happen. We use the Subject plus the modal "WOULD" followed by a consequence (in base infinitive, obviously. We already know that, right?).
    Structure WOULD a consequence (in base infinitive).
    Example I would go everywhere on foot.

We can also explain the consequences with modal verbs of obligation or prohibition. Since we have to use WOULD (a past tense), we can't use MUST or CAN.

Examples:

  • If I didn't have a bike, I would have to go everywhere on foot.
  • If I had a bike, I could go everywhere on foot.
  • If my parents were poor, they wouldn't be able to give me an allowance.

Remember!

When we use first and second conditional, we can switch the order of the two parts. We only put a comma between them if the action goes before the consequence.

If I had a bike, I could go everywhere on foot = I could go everywhere on foot if I had a bike

Think about it...

Both first and second conditional can be used to share possible solutions to conflicts. Using the first conditional means that we are speaking about a very real possibility. Using the second conditional means that you are speaking about possibilities that you are just thinking and sharing. Think about it when you want to be clear about your opinion.

We can use reported speech to share actions and consequences that other people shared with you before. If we pass the first conditional to reported speech, we can change the tenses to the past. If we pass the second conditional to reported speech, we don't need to make any change to the tenses.

Try to use indirect speech to share the following sentences with other people:

  • Your dad says "If we entered a war, many innocent people would die".
  • Your teacher says "If I don't set the test this friday, you will be stressed during the weekend".

Feedback

There are several possibilities:

  • My dad said if we enter a war, many innocent people will die.
  • My dad said if we entered a war, many innocent people would die.
  • My teacher said If he didn't set the test that friday, we would be stressed during the weekend.

Choose the best conditional

Question

1. Which option should we use if we are sure that an action or a consequence will happen?

Answers

If Spain starts a war against the USA, many innocent people will die.

If Spain started a war against the USA, many innocent people would die.

Feedback

Correct! We are saying that we really think that a war can happen.

Incorrect. Here we would be speaking of the consequences of an imaginary war.

Solution

  1. Correct Option (Feedback)
  2. Wrong (Feedback)

Question

2. What is the correct option?

Answers

If humans have wings, we will be able to fly.

If humans had wings, we would be able to fly.

Feedback

This option would only be possible if you really believe that, in the future, humans can have wings.

Correct! It's an obviously imaginary situation.

Solution

  1. Wrong (Feedback)
  2. Correct Option (Feedback)

Question

3. What is the correct option?

Answers

I would die if you won't love me.

I would die if you didn't love me.

Feedback

Incorrect! Don't you think this sentence sounds weird?

Correct! This is the only sentence that is well written.

Solution

  1. Wrong (Feedback)
  2. Correct Option (Feedback)

Question

4. What is the best option?

Answers

If we train a lot, I am sure we will win the game.

If we train a lot, I am sure we won't win the game.

Feedback

Correct! Makes sense, doesn't it? If you believe that you won't win even if you train a lot, you should say, for example:

Even if we train a lot, I am sure we won't win the game.

We have to write well, but we also need to think about the meaning. Do you really think this sentence makes sense?

Solution

  1. Correct Option (Feedback)
  2. Wrong (Feedback)

Question

5. What is the best option to tell a friend about the history of Ghandi?

Answers

Ghandi believed that, even if he didn't use violence, India would become independent.

Ghandi believed that, even if he doesn't use violence, India will become independent.

Feedback

Correct! This is the only possible option.

Incorrect. We are speaking about an old prediction of an event that already happened. This one is tricky, I know.

Solution

  1. Correct Option (Feedback)
  2. Wrong (Feedback)

Question

6. Are the two following sentences correct?

  1.  The president said we will go to war if they attack us.
  2.  The president said that, if they attacked us, we would go to war.

Answers

Yes.

No.

Feedback

Exactly! Both are possible. One is saying we will probably be attacked. The other one is saying the consequence of something that hopefully won't happen.

Are you sure? Read the question well...

Solution

  1. Correct Option (Feedback)
  2. Wrong (Feedback)
